# 🚀 VisionBoard AI

Your AI-powered productivity OS. Plan smarter, focus deeper, and review progress more clearly.

[![Built with VisionBoard AI](https://img.shields.io/badge/Built%20with-VisionBoard%20AI-4F46E5?style=for-the-badge)](https://visionboard.doc.dahir.dev)

## ⚡️ Features

- 🤖 **GPT-Powered Insights**
  - Weekly progress reports
  - Task suggestions & optimization
  - Smart priority sorting

- 📅 **Calendar Integration**
  - Google Calendar sync
  - Drag & drop scheduling
  - Recurring tasks support

- 🎯 **Focus Tools**
  - Pomodoro timer
  - Focus statistics
  - Distraction blocking

- 📊 **Progress Tracking**
  - Beautiful analytics
  - PDF report export
  - Task completion trends

- 🎨 **Modern UI/UX**
  - Dark mode support
  - Responsive design
  - Glassmorphism effects

## 🧠 Tech Stack

- **Frontend**
  - Vue 3 + TypeScript
  - Tailwind CSS
  - Pinia (State)
  - Vite (Build)

- **Backend**
  - Supabase (Auth + DB)
  - Edge Functions
  - Real-time sync

- **Integrations**
  - OpenAI GPT-4
  - Google Calendar API
  - Chart.js

## 🚀 Quick Start

```bash
# Clone the repo
git clone https://github.com/yourusername/visionboard-ai.git

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env

# Start development server
npm run dev
```
